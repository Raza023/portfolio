# 🌐 Hassan Raza — Portfolio Website

> A modern, responsive personal portfolio showcasing my skills, projects, and professional experience as a **Java & Spring Boot Software Engineer**.

[![Live Portfolio](https://img.shields.io/badge/🌐-Visit%20My%20Portfolio-6d28d9?style=for-the-badge&logo=github)](https://raza023.github.io/portfolio/)
---

## ✨ Features

- 🌑 **Dark / Light Mode** — Toggle between themes with preference saved to `localStorage`
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop screens
- 🎨 **Glassmorphism UI** — Premium frosted-glass cards with smooth hover animations
- 🔼 **Scroll to Top** — Floating action button that appears on scroll
- ⚡ **AOS Animations** — Scroll-triggered entrance animations throughout the page
- 🔗 **Live Project Links** — Direct links to GitHub repositories

---

## 🛠️ Built With

| Technology | Purpose |
|---|---|
| HTML5 & CSS3 | Structure & Styling |
| Tailwind CSS | Utility-first CSS framework |
| Feather Icons | Lightweight SVG icon library |
| AOS.js | Scroll animation library |
| Google Fonts (Outfit) | Modern typography |

Just open `index.html` in your browser — no build tools or dependencies needed.

```bash
# Clone the repository
git clone https://github.com/Raza023/portfolio.git

# Open in browser
open index.html
```
